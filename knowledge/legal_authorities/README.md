# Settlement and C Permit Legal Authorities

This folder contains legal authorities, guidance, case law, policy materials, and structured notes relevant to settlement and C permit applications.

## Scope

Use this folder for materials concerning:

- Permanent residence / settlement
- C permit eligibility
- Continuous residence
- Absences
- Integration requirements
- Language requirements
- Suitability and character issues
- Family members applying for settlement or C permits

## Folder structure

- `core/` — central authorities that are repeatedly relevant across settlement and C permit matters.
- `broad/` — high-level principles, general statutory frameworks, policy overviews, and background materials.
- `route_specific/` — authorities relevant to a specific immigration route, category, or applicant type.
- `issue_specific/` — authorities organised by discrete legal or factual issue.

## Inclusion criteria

Add a document here only if it is directly relevant to settlement or C permit work. Where a source is relevant to multiple areas, place it in the most specific applicable folder and cross-reference it elsewhere if needed.

## File naming

Use clear, descriptive filenames

Example:

'legal_authority_core_swiss_c_permit_framework_pack'
