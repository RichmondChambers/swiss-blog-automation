# Route-Specific Settlement and C Permit Authorities

This folder contains authorities and structured notes relevant to specific immigration routes, applicant categories, or sponsor categories in settlement and C permit matters.

## Scope

Use this folder for materials concerning:

- Settlement following different residence routes
- EU/EFTA nationals
- Third-country nationals
- Spouses and family members
- Refugees and temporarily admitted persons, where relevant
- Students and former students, where relevant
- Employment-based residence routes
- Cantonal variations affecting specific routes
- Early C permit eligibility for specific categories

## Inclusion criteria

Add a document here only if it is primarily relevant to a particular route, category, or applicant type.

If the authority applies generally across settlement and C permit work, place it in `core/` or `broad/` instead.

## File naming

Use clear, descriptive filenames.

Example:

`legal_authority_route_specific_swiss_c_permit_eu_efta_nationals`
