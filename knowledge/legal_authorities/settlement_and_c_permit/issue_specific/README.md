# Issue-Specific Settlement and C Permit Authorities

This folder contains authorities and structured notes organised by discrete legal, factual, or evidential issues arising in settlement and C permit matters.

## Scope

Use this folder for materials concerning:

- Absences from Switzerland
- Breaks in residence
- Continuous residence
- Integration evidence
- Language evidence
- Financial independence
- Public order and security issues
- Criminality or character concerns
- Debt, social assistance, or enforcement issues
- Cantonal discretion
- Refusals and remedies
- Evidence requirements

## Inclusion criteria

Add a document here only if it is primarily relevant to a specific issue rather than to the settlement or C permit framework as a whole.

Where an authority applies generally across multiple issues, place it in `core/` and cross-reference it from the relevant issue-specific note if needed.

## File naming

Use clear, descriptive filenames.

Example:

`legal_authority_issue_specific_c_permit_absences_continuous_residence`
