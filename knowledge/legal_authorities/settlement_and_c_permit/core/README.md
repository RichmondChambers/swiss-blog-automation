# Core Settlement and C Permit Authorities

This folder contains central legal authorities and structured notes that are repeatedly relevant across settlement and C permit matters.

## Scope

Use this folder for core materials concerning:

- Statutory provisions governing settlement and C permits
- Principal federal guidance
- Core eligibility frameworks
- Standard residence-period requirements
- Continuous residence principles
- Integration requirements
- Language requirements
- Suitability, public order, and character considerations
- Key interpretive authorities used across multiple settlement and C permit issues

## Inclusion criteria

Add a document here only if it is likely to be relevant across a wide range of settlement or C permit matters.

Do not use this folder for narrow issue-specific materials unless they have general application across settlement and C permit work.

## File naming

Use clear, descriptive filenames.

Example:

`legal_authority_core_swiss_c_permit_framework_pack`
